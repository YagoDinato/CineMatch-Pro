# 🎬 CineMatch Pro - CineLog

Aplicação web para registrar e organizar filmes, séries, documentários e animações que você já assistiu.

## Funcionalidades

- Cadastro de obras com título, tipo, gênero, ano, duração, nota e sinopse
- Catálogo com filtros por tipo e gênero
- Estatísticas (total de obras, horas assistidas, nota média)
- Trailers do YouTube integrados via modal
- Armazenamento local (localStorage) — seus dados ficam no seu navegador

## Tecnologias

- HTML5
- CSS3 (variáveis, grid, flexbox, gradientes)
- JavaScript (ES6+, classes, localStorage)

## Estrutura do projeto
cinelog/

├── index.html      # Página de cadastro

├── catalogo.html   # Página de catálogo

├── css/

│   └── style.css

├── js/

│   ├── main.js      # Classe Filme + lógica do formulário

    └── display.js   # Renderização e filtros do catálogo


## Como usar

Basta abrir o `cinelog/index.html` em um navegador — não requer servidor ou instalação.
